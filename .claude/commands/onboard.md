# Workspace Onboarding Context Protocol

## Scanning Blueprint

1. **Map Repository Topography**
   - Execute `git ls-files` to locate all currently tracked development layers and modules.

2. **Parse Core Architecture Documentation**
   - Scan `CLAUDE.md`, `PRD.md`, and allied environment parameters to align configuration layers.
   - Audit code server entry points, active requirements profiles, and runtime parameters.
   - Review Supabase relational mappings and system logic definitions.

3. **Evaluate Workspace Integrity State**
   - Execute `git status` and `git log -10 --oneline` to locate current active branch trajectories and recent implementation logs.

## Output Matrix

Deliver a crisp onboarding baseline summary mapping:
- The fundamental mission framework of this specific application instance
- The current operating dependencies and tool libraries (FastAPI + `google-genai` + Uvicorn)
- The global layout patterns of frontend and backend modules
- The target working branches alongside recent code changes
