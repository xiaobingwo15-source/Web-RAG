# UI Redesign: WhatsApp-Inspired Chat + Warm Login

## Goal
Replace the dark industrial/PCB theme with:
1. **Login page**: Warm, clear, stable, safe â€” no electronic elements
2. **Admin + Client pages**: WhatsApp-inspired layout and UI

## Research Summary

### WhatsApp Key Design Elements
- **3-panel layout**: Sidebar (chat list, 340px) â†’ Main chat â†’ Info panel
- **Colors**: Teal green `#00A884` primary, `#F0F2F5` bg, `#FFFFFF` cards, `#D9FDD3` outgoing bubbles
- **Typography**: System fonts (Segoe UI / SF Pro), 14-16px body, clean weights
- **Chat bubbles**: 8px radius, 65% max-width, outgoing=green tint, incoming=white
- **Input area**: Clean white field, 8px radius, icon buttons left/right
- **Spacing**: 72px chat list items, 59px headers, generous padding
- **Feel**: Minimal, calm, professional, trustworthy

### Current Problems
- Dark `#111316` background with dot grid (circuit board)
- `.crosshair-corner` PCB decorations
- Industrial blue `#9fcaff` / `#007acc` primary
- "IE Industrial Electronics" branding throughout
- Login has industrial background image + mouse-tracking glow
- Chat bubbles use dark muted colors

### Typography
- Font: `Segoe UI, system-ui, -apple-system, sans-serif` (system fonts like WhatsApp)
- Body: 14px, weight 400
- Headings: 16-17px, weight 500
- Timestamps: 12px, weight 400
- Line height: 1.35-1.4

### Border Radius
- Bubbles: 8px
- Inputs: 8px
- Avatars: 50%
- Badges: 9999px (pill)
- Cards: 12px (slightly softer than WhatsApp for warmth)

## Implementation Plan

### Phase 1: Theme Foundation (`index.css`)
**File**: `frontend/src/index.css`

1. Replace all `--color-*` tokens with warm light palette
2. Remove dot grid background pattern from `body`
3. Remove `.crosshair-corner` class
4. Add WhatsApp chat color tokens
5. Update `.chat-markdown` for light theme
6. Update radius tokens (bump `--radius-lg` to 12px)
7. Change font stack to system fonts

### Phase 2: Login Page Redesign (`LoginPage.tsx`)
**File**: `frontend/src/pages/LoginPage.tsx`

**Remove:**
- Industrial background image
- Mouse-tracking radial glow effect
- Four corner accent divs
- "IE Industrial Electronics" branding
- Uppercase tracking-widest labels

**New design:**
- Clean warm cream background (`#F5F1EB`)
- Centered card (max-w-md) with white bg, subtle shadow, 12px radius
- App logo/name at top (generic, configurable)
- Friendly heading: "Welcome back" (not "Portal Access")
- Email + password fields with clean styling
- "Sign in" button in WhatsApp teal (`#00A884`)
- "Forgot password?" link below
- Minimal footer
- Gentle fade-in animation

### Phase 3: Chat Page WhatsApp Layout (`ChatPage.tsx`)
**File**: `frontend/src/pages/ChatPage.tsx`

**New layout (WhatsApp 2-panel):**
- Left sidebar (w-80-96): Thread list with WhatsApp-style items
  - Search bar at top
  - Thread list: avatar circle + title + last message preview + timestamp
  - Active thread highlighted with `#F0F2F5`
  - Hover state `#F5F6F6`
  - User info + logout at bottom
- Right panel: Active chat
  - Chat header: thread title, "online" status, action icons
  - Chat area: `#EFEAE2` background
  - Messages: WhatsApp bubble style
  - Input area: clean white, bottom-fixed

### Phase 4: Chat Bubbles Redesign (`ChatMessage.tsx`)
**File**: `frontend/src/components/ChatMessage.tsx`

**WhatsApp bubble style:**
- Outgoing (user): `#D9FDD3` bg, right-aligned, 65% max-width, 8px radius
- Incoming (assistant): `#FFFFFF` bg, left-aligned, 65% max-width, 8px radius
- Timestamp inside bubble, bottom-right, 11px `#667781`
- Small tail/arrow on first message in group
- Sources panel: subtle card below bubble
- Feedback buttons: minimal, shown on hover

### Phase 5: Chat Input Redesign (`ChatInput.tsx`)
**File**: `frontend/src/components/ChatInput.tsx`

**WhatsApp input style:**
- Container: `#F0F2F5` bg, fixed bottom
- Text input: white bg, 8px radius, no border, 15px font
- Left: attachment/emoji icons (gray `#54656F`)
- Right: send button (teal `#00A884` when active, gray when empty)
- RAG toggle: subtle pill button, not prominent

### Phase 6: Admin Page WhatsApp Layout (`AdminPage.tsx`)
**File**: `frontend/src/pages/AdminPage.tsx`

**WhatsApp-inspired admin:**
- Left sidebar (w-80): Navigation + thread list
  - Profile avatar + "Admin" label at top
  - Tab pills (Chats/Users/Settings/Evals)
  - Thread list with WhatsApp-style items
  - **Document upload at bottom of sidebar (collapsible)** â€” collapsed by default, expand with chevron
- Right panel: Content area
  - Header bar with title + actions
  - Clean white content area
  - Tables with WhatsApp-style minimal borders
  - Status badges as pill shapes

### Phase 7: Sidebar Components (`ChatSidebar.tsx`, `ChatHistoryPanel.tsx`)
**Files**: `frontend/src/components/ChatSidebar.tsx`, `ChatHistoryPanel.tsx`

- Redesign to match WhatsApp sidebar style

- Search bar: `#F0F2F5` bg, rounded, search icon
