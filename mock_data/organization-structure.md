## Department Breakdown

### Engineering Division (380 employees)
**Head**: Dr. Markus Braun (CTO)

- **Embedded Systems Team** (95 engineers) — PLC firmware, real-time OS, device drivers
- **Cloud Platform Team** (80 engineers) — SmartEdge cloud services, API gateway, data pipeline
- **Edge Computing Team** (60 engineers) — Edge runtime, protocol adapters, local inference
- **Quality Assurance Team** (45 engineers) — Test automation, hardware validation, compliance testing
- **DevOps & Infrastructure** (35 engineers) — CI/CD, monitoring, cloud infrastructure
- **Technical Documentation** (15 writers) — User manuals, API docs, knowledge base
- **Cybersecurity Team** (50 engineers) — IEC 62443 compliance, penetration testing, secure boot

### R&D Division (220 employees)
**Head**: Dr. Yuki Tanaka

- **Robotics Lab** (55 researchers) — Kinematics, force control, path planning, human-robot collaboration
- **Machine Vision Lab** (50 researchers) — Deep learning for inspection, 3D reconstruction, OCR
- **AI & Analytics Lab** (60 researchers) — Predictive maintenance models, anomaly detection, digital twins
- **Sensor Technology Lab** (35 researchers) — MEMS design, signal processing, wireless protocols
- **Advanced Prototyping** (20 engineers) — 3D printing, rapid prototyping, pilot production

### Sales & Marketing Division (180 employees)

- **International Sales** (45 staff) — Europe (18), North America (15), Japan/Korea (12)
- **Marketing & Brand** (25 staff) — Digital marketing, events, content, PR
- **Pre-Sales Engineering** (30 staff) — Solution design, POC support, technical demos
