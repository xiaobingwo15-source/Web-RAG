## Key Achievements in 2025

1. **SmartEdge 3.0 Launch**: Major platform release with native AI inference at the edge, reducing cloud dependency by 60% for latency-sensitive applications.

2. **Automotive Sector Expansion**: Won contracts with 3 major OEMs (BYD, BMW Spartanburg, Toyota Kentucky) for factory-wide automation deployments worth RMB 320M over 3 years.

3. **IE-Predict Growth**: Predictive maintenance module grew 65% YoY, now monitoring 45,000+ machines across 200+ factories. Customer-reported unplanned downtime reduced by 73% on average.

4. **Patent Portfolio**: Filed 128 new patents in 2025 (total: 612). Key areas: edge AI inference (32 patents), industrial cybersecurity (18 patents), robotic path planning (15 patents).

5. **Strategic Partnership**: Signed a joint development agreement with Microsoft Azure for native SmartEdge integration with Azure IoT Hub, available Q2 2026.
